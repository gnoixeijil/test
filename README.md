# vue-case

------

#### 介绍

vue-case 是基于 [vue-element-admin](http://panjiachen.github.io/vue-element-admin) 开发的小工具，里面包含了Echart，svg，富文本编辑器TinyMCE等

------



#### 项目结构

vue 2.6.10 + vue-router 3.0.6 + vuex 3.1.0 + axios + echarts 5.3.3 + tinymce 5.1.0 

------

####  安装教程

```
<!--安装依赖-->
`npm install`
<!--项目启动-->

`npm run dev`

<!--访问项目-->

http://localhost:9529

账号：admin

密码：123456
```
